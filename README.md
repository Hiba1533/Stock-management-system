Stock_management_system
A full-stack inventory management application designed to help businesses track stock movements, monitor low inventory levels, and generate automated reports. This project combines Python backend logic with a web-based frontend for seamless management.


Features
- Track Inventory Movements: Add, update, and remove products with real-time stock updates.
- Low-Stock Alerts: Get notified when product quantities fall below predefined thresholds.
- Automated PDF Reports: Generate detailed reports of stock movements and inventory levels.
- Data Analysis & Visualization: Analyze inventory trends using charts and graphs for smarter decision-making.


Technologies Used
- Backend: Python (Pandas, NumPy, Matplotlib, Seaborn, FPDF)
- Frontend: HTML, CSS, JavaScript
- Database: MySQL
- Development Environment: VS Code


Database schema
The full database schema is available in [`stock sys.sql`](stocksys.sql). 


Usage
- Login with the admin account (from stocksys.sql).
- Add categories, products, suppliers, and customers.
- Record purchase orders and sales orders.
- Monitor stock levels and receive low-stock alerts.
- Generate PDF reports and analyze inventory trends.
